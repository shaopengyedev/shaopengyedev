# 第一定律：`!(A && B) == (!A || !B)`

- `!(A && B)`：考虑，除了`A && B`[^1]，所有情况：
    - `!A`：1）`!A && B`；2）`!A && !B`。
    - `!B`：1）`A && !B`。
- 具体使用：
    - `if (A && B) { ... } else { if (!A) { ... } else { ... } }`
    - `while (A && B) { ... } if (!A) { ... } else { ... }`
