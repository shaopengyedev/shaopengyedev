# 传入类型实参

1. **只要使用[^1][^2]泛型类，就要传入[^3][^4][[05-001-02-02-01|类型实参]]。**
2. *声明什么泛型 => 传入什么类型实参*：
    - 声明`泛型 extends 上界`，传入`上界`或`上界子类`，可以调用`上界方法`。 ^000ebf
3. 根据*[[05-001-02-02-02.java|具体类型]]*：
    1. “写”^[泛型，用来表示，方法的参数类型。]：编译时，**检查实参类型（具体类型、具体类型子类）**。 ^5ba299
    2. “读”[^5]：编译时，**强制类型转换（具体类型[^6]）**。
