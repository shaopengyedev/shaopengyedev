# 传入类型实参

1. **只要使用[^1]泛型类，就要传入[^2][[05-001-02-02-01|类型实参]]。**
2. *声明什么泛型 => 传入什么类型实参*
    - 声明`泛型 extends 上界`，传入`上界`或`上界子类`，可以调用`上界方法`。 ^000ebf
3. 根据*[[05-001-02-02-02.java|具体类型]]*：
    1. “写”[^3]：编译时，**检查实参类型（具体类型、具体类型子类）**。
    2. “读”[^4]：编译时，**强制类型转换（具体类型）**。

[^1]: 上界、泛型父类、泛型类对象。

[^2]: 在声明泛型类对象时、在调用泛型方法时。

[^3]: 泛型用来表示方法的参数类型。

[^4]: 泛型用来表示方法的返回值类型。
