# `arraycopy(src, srcPos, dest, destPos, length)`

```java
public static native void arraycopy(Object src, int srcPos,
                                    Object dest, int destPos,
                                    int length);
```

## 高效拷贝数组元素

- 高效：将`src[srcPos]` - `src[srcPos + length - 1]`，*整体拷贝*到`dest[destPos]` - `dest[destPos + length - 1]`。
    - 左闭右开。[^1]
- 支持`src == dest`。[^2]
- 支持`length == 0`。

[^2]: `src` -> 临时数组 -> `dest`
