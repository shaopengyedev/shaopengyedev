# `remove(index)`

```java
@Override
public E remove(int index) {
    E ret = get(index);

    // index+1 - size-1 -> index - size-2
    // for (int i = index + 1; i < size; i++) {
    //     elementData[i - 1] = elementData[i];
    // }
    System.arraycopy(elementData, index + 1, elementData, index, size - (index + 1));
    elementData[size - 1] = null;
    size--;

    return ret;
}
```
