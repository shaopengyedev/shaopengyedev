# `removeLast()`

```java
@SuppressWarnings("unchecked")
@Override
public E removeLast() {
    int t = (tail - 1) & (elements.length - 1);
    E ret = (E)elements[t];
    if (ret == null) {
        throw new NoSuchElementException();
    }

    elements[t] = null;
    tail = t;

    return ret;
}
```
