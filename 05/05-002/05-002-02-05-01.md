# 物理结构：离散存储

## 节点 = 元素`e` + 前驱节点`prev` + 后继节点`next`

- 头节点`head`：`head.prev == null`
    - 不一定是起始节点。[^1]
- 尾节点`tail`：`tail.next == null`
- 用户元素个数`size`：
    - `size == 0`：`head == null && tail == null`
    - `size == 1`：`head == oneNode && tail == oneNode`
