# `addFirst(e)`

```java
@Override
public void addFirst(E e) {
    Objects.requireNonNull(e);

    // head：第一个用户元素的索引
    head = (head - 1) & (elements.length - 1);
    elements[head] = e;

    if (head == tail) {
        doubleCapacity();
    }
}
```
