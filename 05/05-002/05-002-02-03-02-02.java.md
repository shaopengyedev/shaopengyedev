# `remove(index)`

```java
@Override
public E remove(int index) {
    checkElementIndex(index);

    Node<E> prev = null;
    Node<E> node = head;
    for (int i = 1; i <= index; i++) {
        prev = node;
        node = node.next;
    }

    E ret = node.e;
    node.e = null;
    Node<E> next = node.next;
    node.next = null;
    // prev == null => node == head
    if (prev == null) {
        head = next;
    }
    else {
        prev.next = next;
    }
    size--;
    return ret;
}
```
