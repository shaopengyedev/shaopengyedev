# `add(index, element)`

```java
@Override
public void add(int index, E element) {
    checkPositionIndex(index);

    Node<E> prev = null;
    Node<E> next = head;
    for (int i = 1; i <= index; i++) {
        prev = next;
        next = next.next;
    }

    Node<E> node = new Node<>(element);
    node.next = next;
    if (prev == null) {
        head = node;
    }
    else {
        prev.next = node;
    }
    size++;
}
```
