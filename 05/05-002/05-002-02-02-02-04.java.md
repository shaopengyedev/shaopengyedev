# `toString()`

```java
@Override
public String toString() {
    StringBuilder res = new StringBuilder();

    res.append('[');
    int i = 0;
    while (i < size) {
        res.append(elementData[i]);

        i++;
        if (i < size) {
            res.append(", ");
        }
        /// 没有进入循环，不会输出']'
        // else {
        //     res.append(']');
        // }
    }
    res.append(']');

    return res.toString();
}
```
