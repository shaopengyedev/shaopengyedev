# `removeFirst()`

```java
@SuppressWarnings("unchecked")
@Override
public E removeFirst() {
    E ret = (E)elements[head];
    if (ret == null) {
        throw new NoSuchElementException();
    }

    elements[head] = null;
    head = (head + 1) & (elements.length - 1);

    return ret;
}
```
