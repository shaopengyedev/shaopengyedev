# `remove(index)`

```java
@Override
public E remove(int index) {
    checkElementIndex(index);

    Node<E> node;
    // 0 - size-1
    int mid = (size - 1) >> 1;
    if (index <= mid) {
        node = head;
        for (int i = 1; i <= index; i++) {
            node = node.next;
        }
    }
    else {
        node = tail;
        for (int i = size - 2; i >= index; i--) {
            node = node.prev;
        }
    }

    E ret = node.e;
    node.e = null;
    Node<E> prev = node.prev;
    node.prev = null;
    Node<E> next = node.next;
    node.next = null;
    if (next == null) {
        tail = prev;
    }
    else {
        next.prev = prev;
    }
    if (prev == null) {
        head = next;
    }
    else {
        prev.next = next;
    }
    size--;
    return ret;
}
```
