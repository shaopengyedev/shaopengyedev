# Linux文献 网络管理和设置

- 通信： #gemini
    - 局域网内[^1]通信：例如局域网A：
        - 节点x <-交换机A-> 节点y
    - 跨局域网通信：例如局域网A和局域网B：
        - 节点x <-交换机A-> 网关[^2]A
        - 网关A <-广域网[^3]-> 网关B
        - 网关B <-交换机B-> 节点y
- 物理机 <-通信-> 虚拟机
    - [ ] VMware：编辑 > 虚拟网络编辑器 > VMnet8
        - [ ] 子网IP：`192.168.10.0`
        - [ ] 子网掩码：`255.255.255.0`
        - [ ] NAT设置
            - [ ] 网关IP：`192.168.10.2`
    - [ ] 物理机：控制面板 > 网络和Internet > 网络和共享中心 > 更改适配器设置 > VMware Network Adapter VMnet8 > 状态 > 详细信息
        - [ ] IPv4地址：`192.168.10.1` #tocheck
    - [ ] 虚拟机：
        - [ ] 配置静态IP地址：
            - [ ] `vim /etc/sysconfig/network-scripts/ifcfg-ens33` #tocheck
                - [ ] `BOOTPROTO="static"`
                - [ ] `ONBOOT="yes"`
                - [ ] `IPADDR=192.168.10.100`
                - [ ] `GATEWAY=192.168.10.2`
                - [ ] `DNS1=192.168.10.2`
            - [ ] `systemctl restart network`
            - [ ] 验证：
                - [ ] 虚拟机：`ping 192.168.10.1`、`ping www.baidu.com`
                - [ ] 物理机：`ping 192.168.10.100`
        - [ ] 配置主机名：
            - [ ] `vim /etc/hostname` #tocheck
                - [ ] `hadoop100`
            - [ ] 重启

[^1]: 同一个子网/网段。

[^2]: 路由器。

[^3]: 互联网。
