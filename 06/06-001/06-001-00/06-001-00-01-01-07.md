# Linux文献 网络管理和设置

- 通信：
    - 局域网内[^1]通信
        - 局域网A：节点x <-交换机A-> 节点y
    - 跨局域网通信
        - 局域网A：节点x <-交换机A-> 网关[^2]A
        - 网关A <-广域网[^3]-> 网关B
        - 局域网B：网关B <-交换机B-> 节点y
- 物理机 <-通信-> 虚拟机
    - [ ] VMware：
        - [ ] 配置虚拟网络：
            - [ ] 编辑 > 虚拟网络编辑器 > VMnet8（NAT模式）
                - [ ] 子网IP：`192.168.10.0`
                - [ ] 子网掩码：`255.255.255.0`
                - [ ] NAT设置
                    - [ ] 网关IP：`192.168.10.2`
    - [ ] 物理机：
        - [ ] 配置IP地址：
            - [ ] 控制面板 > 网络和Internet > 网络和共享中心 > 更改适配器设置 > VMware Network Adapter VMnet8 > 状态 > 属性 > Internet协议版本4 (TCP/IPv4)
                - [ ] 使用下面的IP地址
                    - [ ] IP地址：`192.168.10.1`
                    - [ ] 子网掩码：`255.255.255.0`
                    - [ ] 默认网关：`192.168.10.2`
                - [ ] 使用下面的DNS服务器地址
                    - [ ] 首选DNS服务器：`192.168.10.2`
        - [ ] 配置IP地址映射（“域名”）：
            - [ ] `vim /c/Windows/System32/drivers/etc/hosts`
                - [ ] `192.168.10.100 hadoop100`
                - [ ] ……
                - [ ] `192.168.10.108 hadoop108`
    - [ ] 虚拟机：
        - [ ] 配置IP地址：
            - [ ] `vim /etc/sysconfig/network-scripts/ifcfg-ens33`
                - [ ] 静态分配[^4]IP地址：`BOOTPROTO="static"`
                - [ ] 在系统启动时自动激活网卡：`ONBOOT="yes"`
                - [ ] IP地址：`IPADDR=192.168.10.100`
                - [ ] 网关：`GATEWAY=192.168.10.2`
                - [ ] 域名解析器：`DNS1=192.168.10.2`
            - [ ] 重启网络服务：`systemctl restart network`
            - [ ] 验证：
                - [ ] 虚拟机：`ping 192.168.10.1`、`ping www.baidu.com`
                - [ ] 物理机：`ping 192.168.10.100`、`ping hadoop100`
        - [ ] 配置主机名和IP地址映射[^5]：
            - [ ] `vim /etc/hostname`
                - [ ] `hadoop100`
            - [ ] `vim /etc/hosts`
                - [ ] `192.168.10.100 hadoop100`
                - [ ] ……
                - [ ] `192.168.10.108 hadoop108`
            - [ ] `shutdown -r now`

[^1]: 同一个子网/网段。

[^2]: 路由器。

[^3]: 互联网。

[^4]: IP地址固定，方便节点之间的通信。

[^5]: 方便节点之间的通信。
