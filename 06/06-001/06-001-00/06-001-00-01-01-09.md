# 开关机重启命令 13 视频day01 Linux文献

- 服务器7X24小时跑服务 => 在特殊情况下，不得已才会关机。
- `sync`：立即将缓冲区的数据写入磁盘。
    - 迟写：数据 -> 缓冲区 -达到阈值-> 磁盘
        - => 极大地提高磁盘写入数据的效率
        - => 可能丢失数据
    - 先`sync`，再`shutdown`。
- `shutdown -h now | MINS`：
    - `-h`：关机。
    - `now`：立即执行。
    - `MINS`：`MINS`分钟后执行。
- `shutdown -r now | MINS`：
    - `-r`：重启。
