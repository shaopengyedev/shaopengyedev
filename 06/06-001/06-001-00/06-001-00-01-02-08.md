# 常用命令-文件目录权限管理类 12-14 视频day02 Linux文献

- 文件的操作权限：
    - 目录：
        - 读：`ls`
        - 写：`mkdir`、`touch`、`cp`、*`rm`*、`mv`
            - 删除一个文件的前提：具有其父目录的写权限。
        - 执行：*`cd`*
    - 文件：
        - 读：`cat`、`less`、`head`、`tail`
        - 写：`vim`、`>`、`>>`
        - 执行：`bash`
- `ls -l`
    - 文件类型：
        - `d`：表示目录。
        - `-`：表示文件。
        - `l`：表示软链接。
    - 文件的操作权限列表 = 所属主的读写执行权限 + 所属组的读写执行权限 + 其他用户的读写执行权限
        - `r`：表示具有读权限。
        - `w`：表示具有写权限。
        - `x`：表示具有执行权限。
        - `-`：表示没有权限。
    - 链接数：
        - 目录：子目录数量。
        - 文件：硬链接数量。
    - 文件所属主：默认为创建文件的用户。
    - 文件所属组：默认为创建文件的用户，其用户组。
    - 文件大小
        - 目录：固定4096，需要使用`du`计算物理大小。
    - 文件最后修改时间
    - 文件名

---

- 操作的用户：是目标文件的什么用户 => 具有什么操作权限：
    1. 是目标文件的所属主 => 具有所属主的操作权限
    2. 是目标文件的所属组的组内用户 => 具有所属组的操作权限
    3. 是其他用户 => 具有其他用户的操作权限

---

- `chmod [-R] MODE PATH`：修改文件或目录的操作权限。
    - `-R`：递归修改。
    - `MODE`：例如`u=rwx,g=rx,o=rx`或`755`。
- `chown [-R] NEW_USER:NEW_GROUP PATH`：修改文件或目录的所属主和所属组。
    - `-R`：递归修改。
