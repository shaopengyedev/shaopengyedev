# 配置IP地址

- [ ] `vi /etc/sysconfig/network-scripts/ifcfg-ens33`
    - [x] 静态分配[^1]IP地址：`BOOTPROTO=static`
    - [x] 在系统启动时自动激活网卡：`ONBOOT=yes`
    - [x] IP地址：`IPADDR=192.168.10.100`
    - [x] 网关：`GATEWAY=192.168.10.2`
    - [x] 域名解析器：`DNS1=192.168.10.2`
- [ ] 停止NetworkManager服务，并禁用开机自启：[^2]
    - [x] `systemctl stop NetworkManager`
    - [x] `systemctl disable NetworkManager`
- [ ] 重启网络服务：
    - [x] `systemctl restart network`
- [ ] 验证：
    - [x] 虚拟机：`ping -c 3 192.168.10.1`、`ping -c 3 www.baidu.com`
    - [x] 物理机：`ping 192.168.10.100`、`ping hadoop100`

[^1]: IP地址固定，方便节点之间的通信。

[^2]: NetworkManager服务可能停止network服务。
