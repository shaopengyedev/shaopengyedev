# 配置IP地址

- [ ] `vi /etc/sysconfig/network-scripts/ifcfg-ens33`
    - [ ] 静态分配[^1]IP地址：`BOOTPROTO=static`
    - [ ] 在系统启动时自动激活网卡：`ONBOOT=yes`
    - [ ] IP地址：`IPADDR=192.168.10.100`
    - [ ] 网关：`GATEWAY=192.168.10.2`
    - [ ] 域名解析器：`DNS1=192.168.10.2`
- [ ] 重启网络服务：
    - [ ] `systemctl restart network`
- [ ] 停止NetworkManager服务，并禁用开机自启：[^2]
    - [ ] `systemctl stop NetworkManager`
    - [ ] `systemctl disable NetworkManager`
- [ ] 验证：
    - [ ] 虚拟机：`ping 192.168.10.1`、`ping www.baidu.com`
    - [ ] 物理机：`ping 192.168.10.100`、`ping hadoop100`

[^1]: IP地址固定，方便节点之间的通信。

[^2]: NetworkManager服务可能停止network服务。
