# 临时目录

- 自动删除，超过10天或30天，未变动的文件。
    - 未变动：未读取 || 未修改文件内容 || 未修改文件属性（例如权限）
- 操作权限列表：`rwxrwxrwt.`。
    - `t`：具有执行权限，但不能删除或重命名其他用户的目录或文件。

```bash
[root@hadoop100 ~]# cat /usr/lib/tmpfiles.d/tmp.conf
...
# Clear tmp directories separately, to make them easier to override
v /tmp 1777 root root 10d
v /var/tmp 1777 root root 30d
...
```
