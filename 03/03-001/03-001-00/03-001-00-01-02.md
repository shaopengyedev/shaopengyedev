# 高考英语单词文献：哑音E

- 哑音E：e不发音。
- 触发场景：
    - 原生词结尾：
        - “元音 + 辅音 + e”且重读：前面的元音，发它的字母本音（长音）。[^1]
        - “ue”或“ve”：保证“避免单词以u或v结尾”[^2]。[^3]
        - “ce”或“ge”：改变发音，c发/s/而非/k/，g发/dʒ/而不非/ɡ/。[^4]
        - “se”：避免误以为是复数形式。[^5]
        - “辅音 + l + e”：保证“每个音节必须包含一个元音字母”。[^6]
    - 区分同音词[^7]
- 去e：上述的原生词 + 元音开头的后缀[^8]
    - “ce”或“ge”：保留e来改变c和g的发音。[^9]

[^1]: 例如：bit/bɪt/（短音） -> bite/baɪt/（长音）

[^2]: 极个别外来词除外。

[^3]: 例如：active、give、have、blue。

[^4]: 例如：dance、cage。

[^5]: 例如：house。

[^6]: 例如：table、apple、handle。

[^7]: 例如：by和bye、or和ore。

[^8]: 例如：making。

[^9]: 例如：changeable。
